#  MindEase – Mental Health Care Web Application

MindEase is a full-stack mental health support web application designed to help users assess their emotional well-being and track daily mood patterns. The platform promotes self-awareness through structured questionnaires and interactive mood tracking.

This project demonstrates frontend-backend integration, full-stack development skills, and real-world problem-solving using Python and web technologies.

---

##  Features

-  Mental health self-assessment questionnaire
-  Daily mood tracking system
-  Dynamic UI using JavaScript
-  Backend processing with Python (Flask)
-  Clean and responsive interface
-  User-friendly design

---

##  Tech Stack

- **Backend:** Python
- **Frontend:** HTML5, CSS3, JavaScript
- **IDE:** VS Code
- **Version Control:** Git & GitHub
